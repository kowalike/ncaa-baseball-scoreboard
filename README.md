# NCAA College Baseball Scoreboard
Displays NCAA Division I baseball scores by date. Users can search teams, filter by conference, pick dates via a calendar button or date chips, view live/final game results, and mark a favorite team, with a responsive layout for desktop and mobile.

[Try it now &raquo;](https://kowalike.github.io/ncaa-baseball-scoreboard/)
